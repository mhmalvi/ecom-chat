Enhance your eCommerce store with AI-powered assistance

How to Embed the Chat Widget

Add the following script to your website's HTML to embed the AI chatbot widget:

<script 
  src="https://your-backend-url.com/widget.js" 
  data-api-key="your_store_api_key"
  data-color="#4F46E5"
  data-position="right">
</script>

Configuration Options

Attribute Description Default
data-api-key Your store's API key Required
data-api-url Custom API endpoint URL Current domain
data-color Primary color for the widget #4F46E5
data-position Widget position (right or left) right
